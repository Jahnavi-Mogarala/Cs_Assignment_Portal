Node.js + Express Backend — API Endpoints
POST /api/auth/student/login — Authenticate student
POST /api/auth/faculty/login — Authenticate faculty
GET /api/students — List all students (admin)
POST /api/students — Add student (admin)
GET /api/assignments — Get all assignments
POST /api/assignments — Create assignment
POST /api/submissions/upload — Upload submission (multipart)
GET /api/submissions/:studentId — Student's submissions
GET /api/submissions/all — All submissions (faculty)
POST /api/evaluations — Save evaluation
GET /api/evaluations/:studentId — Student evaluations
GET /api/plagiarism/:assignmentId — Similarity results
GET /api/reports/class — Class-wide report
GET /api/reports/student/:id — Individual report
